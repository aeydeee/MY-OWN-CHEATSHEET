My Own CheatSheet

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

CSS selectors and specificity

Attribute Name Specificity
<div style="..."> Inline Styles Highest
#myId ID Selector High
.myClass Class Selector High
[type="text"] Attribute Selector Medium
p, div, span, etc. Element Selector Low

Common CSS properties

Property Usage Description
color color: red; Sets the text color
background-color background-color: #f0f0f0; Sets the background color of an element
font-size font-size: 16px; Sets the size of the font
font-family font-family: Arial, sans-serif; Sets the font family
font-weight font-weight: bold; Sets the font weight
text-align text-align: center; Sets the horizontal alignment of text within an element
padding padding: 10px; Sets the padding (space) inside an element
margin margin: 20px; Sets the margin (space) outside an element
border border: 1px solid #000; Sets the border properties of an element
width width: 300px; Sets the width of an element
height height: 200px; Sets the height of an element
display display: block; or
display: inline-block; 		Specifies how an element should be displayed
position position: relative; or
position: absolute; 		Specifies the positioning method for an element
float float: left; or
float: right; or
float: none; 		Specifies whether an element should float to the left, right, or none
overflow overflow: hidden; or
overflow: scroll; 		Specifies how overflowing content should be handled
